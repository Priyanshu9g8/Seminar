# Gyan Setu - AI-Powered Educational Platform 🎓✨

Gyan Setu is a modern, multilingual educational platform designed to empower students and teachers through AI-driven learning and secure exam integrity measures.

## 🚀 Key Features

- **AI-Powered Learning**: Personalized AI teaching assistant and automated quiz generation.
- **Exam Integrity**: Robust anti-cheating system with forced Full Screen mode and tab-switching monitoring.
- **Multilingual Support**: Fully localized interface in English, Hindi, and Punjabi.
- **Interactive Dashboards**: Dedicated portals for Students, Teachers, and Admins.
- **Progress Tracking**: Real-time analytics and performance graphs.

## 🛠️ Tech Stack

### Frontend
- **React.js** (Vite)
- **Tailwind CSS** (Styling)
- **i18next** (Multilingual Support)
- **Lucide React** (Icons)

### Backend
- **Java Spring Boot**
- **Maven** (Dependency Management)
- **MySQL** (Database)
- **JWT** (Security & Authentication)

## 📦 Project Structure
```text
gyan-setu/
├── gyan-setu-frontend/     # React Application
└── gyan-setu-backend/      # Spring Boot Application
```

## ⚙️ Setup Instructions

### Backend
1. Navigate to `gyan-setu-backend`.
2. Configure your database in `src/main/resources/application.properties`.
3. Run `mvn spring-boot:run`.

### Frontend
1. Navigate to `gyan-setu-frontend`.
2. Install dependencies: `npm install`.
3. Run dev server: `npm run dev`.
